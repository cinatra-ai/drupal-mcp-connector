# Drupal

Let Cinatra agents read, draft, and publish content on your Drupal sites. Connect one or more Drupal instances and your agents can browse recent nodes, spin up draft revisions, edit fields, and push updates live — including a natural-language path that turns a single instruction into the right sequence of edits.

## Capabilities

- Browse and read recent Drupal nodes from inside an agent flow
- Draft a new node of any configured content type
- Update fields on an existing node through a clean draft revision
- Publish a draft to make it live
- Edit a Drupal node from a plain-language instruction
